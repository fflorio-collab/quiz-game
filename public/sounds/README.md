# Cartella per gli effetti sonori

Metti qui 6 file audio (MP3, brevi < 1s):

- `tick.mp3` — tick del timer
- `correct.mp3` — risposta corretta
- `wrong.mp3` — risposta sbagliata
- `join.mp3` — nuovo giocatore
- `start.mp3` — inizio partita
- `finish.mp3` — fine partita

Se mancano, l'app funziona lo stesso senza audio.

Fonti gratuite:
- https://mixkit.co/free-sound-effects/game
- https://freesound.org
